# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this project, please report it
privately.

- **Do not** open a public GitHub issue
- Email: <olatunbosunoyeleke@hotmail.com>

Please include:
- A description of the issue
- Steps to reproduce
- Potential impact

I will acknowledge receipt and respond as quickly as possible.

## Supported Versions

Only the latest released version is supported with security updates.
